Frequently Asked Questions

What is Hawkpass?

Hawkpass uses a cryptographically secure random number generator based on your mouse movements to generate easy to remember passwords based on the Diceware list.

Entropy

Think of entropy as randomness. In computing, entropy is the randomness collected from hardware sources, mouse movements or randomness generators for use in cryptography.
Entropy is a measure of the uncertainty or randomness of a system. The concept is a difficult one to grasp fully and is confusing, even to experts. Strictly speaking, any given passphrase has an entropy of zero because it is already chosen. It is the method you use to randomly select your passphrase that has entropy. Entropy tells how hard it will be to guess the passphrase itself even if an attacker knows the method you used to select your passphrase. A passphrase is more secure if it is selected using a method that has more entropy.
Entropy is measured in bits. The outcome of a single coin toss -- "heads or tails" -- has one bit of entropy.

Arnold G. Reinhold - Diceware FAQ

True randomness

People don't tend to use password generators because they feel they can come up with very strong "random" password themeselves. But the truth is that we humans are species of patterns and we can't do anything randomly. Even if you think you choose a phrase made from completely random words this phrase is influenced by the probability of a phrase occuring in natural language. [1]

You can also use a computer to generate a password but how secure is that?
Most random numbers generated by computers are pseudo-random i.e. they are generated using a mathematical formula, which is predictable.

The only true randomness is the one that comes from nature. One example of true randomness is atmospheric noise. A software that uses randomness derived from nature is much more secure than a software with computer generated randomness. [2]

Is it safe to use?

Everything is running locally in your browser. None of the passwords generated here are sent anywhere. Once the main page is loaded you can even run the application without an internet connection. This application is just HTML, CSS and Javascript. There is nothing running on a server and no code (jQuery, Bootstrap) is served from a Content Delivery Network (CDN).
